# Snake Game

Snake Game made with HTML, CSS, and JS.

## Getting started

1. Clone this repo.

```bash
git clone https://github.com/jpssantiago/snake-game
```

2. Copy index.html path and paste it into your browser address bar.

3. Enjoy :)
